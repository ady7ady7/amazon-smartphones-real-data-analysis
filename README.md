# Amazon Smartphones Real Data Analysis

A Python data analysis project examining how Apple, Samsung, and Google smartphones lose value on the Amazon US secondary market after launch. Built using real price data from the Keepa API across 63 submodels and ~878,000 weekly price records spanning 2019-2026.
